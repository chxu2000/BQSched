# BQSched
BQSched - Encrypted Code Repository for VLDB 2024 Submission "BQSched: Non-Intrusive Batch Query Scheduling via Reinforcement Learning"

This repository contains the code prepared for the VLDB 2024 submission "BQSched: Non-Intrusive Batch Query Scheduling via Reinforcement Learning". However, it is subject to confidentiality requirements imposed by our collaborator, Huawei. To comply with the conference's reproducibility requirements, we have encrypted the code and shared it in the form of an encrypted archive for the review by the reviewers. If you have the encryption key, please decrypt the compressed file in this repository and access the code.
