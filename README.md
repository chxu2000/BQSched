# BQSched
BQSched - Encrypted Code Repository for ICDE 2025 Submission "BQSched: A Non-intrusive Scheduler for Batch Concurrent Queries via Reinforcement Learning"

Please download the compressed file `BQSched.zip` and decompress it with the encryption key submitted via the Conference Management Toolkit (CMT) to access the codes. You can use common decompression tools like [Windows File Explorer](https://www.microsoft.com/en-us/windows/tips/file-explorer), [7-Zip](https://www.7-zip.org/download.html), or [Bandizip](https://www.bandisoft.com/bandizip/). Currently, these codes are restricted to internal access within the reviewer community, as explained in the response to the Artifact Availability Requirements.
